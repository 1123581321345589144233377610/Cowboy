# Ковбой
Ковбой - это сногсшибательная 2D игра с самым лучшим в мире сюжетом, пророботанными персонажами, замечательной боёвкой и кучей фишек, которые вы можете открыть во время игры!

## Управление:
- W - ВВЕРХ
- S - ВНИЗ
- A - ВЛЕВО
- D - ВПРАВО
- SPACE - ВЫСТРЕЛ (НУЖНО УДЕРЖИВАТЬ)
